OVERVIEW¶

The general objective for this study is to establish the effect of credit management on the profitability of deposit money banks in Nigeria. The specific objective of the study includes to:

  • Examine the effect of loans and advances, nonperforming loans, loan loss provisions on the return on assets (ROA), of deposit money banks in Nigeria.
  • Determine the effect of loans and advances, nonperforming loans, loan loss provisions on the return of equity (ROE) of deposit money banks in Nigeria
  • Ascertain the effect of loans and advance, non-performing loans, loan loss provisions on the return on capital employed (ROCE) of deposit money banks in Nigeria.

DATA OVERVIEW¶

The Dataset is comprises of 60 rows and 9 columns; the columns in the dataset are:

  • ID: Unique count for the dataset, serves as primary key.
  • Bank: The name of the bank in short
  • Year: The relevant accounting year for which the value was calculated/derived
  • ROA: The Return on Asset calculated as Net Income/Total Asset
  • ROE: The Return on Equit calculated as Net Income/Total Equity
  • ROCE: The Return on Capital Employed, calculated as EBIT/(TE + NTL)
  • NPL: Non Performing Loans
  • LA: Loan and Amortization
  • LLP: Loan Loss Provision

RESEARCH QUESTIONS¶

In order to achieve the above study objectives, the research aims at addressing the following questions in relation to the selected banks.

  • To what extent does loans and advances, nonperforming loans, loan loss provisions on the return on assets (ROA) of deposit money banks in Nigeria.
  • What is the effect of loans and advances, nonperforming loans, loan loss provisions on the return of equity (ROE) of deposit money banks in Nigeria.
  • To what extent does loans and advance, non-performing loans, loan loss provisions on the return on capital employed (ROCE) of deposit money banks in Nigeria.

Test of Hypothesis¶

Hypotheses to be tested in this study are stated below in their null forms:

$H_1$: There is no effect of loans and advances, nonperforming loans, loan loss provisions on the return on assets (ROA) of deposit money banks in Nigeria.

$H_2$: There is no effect of loans and advances, nonperforming loans, loan loss provisions on the return on equity (ROE) of deposit money banks in Nigeria.

$H_3$: There is no effect of loans and advances, nonperforming loans, loan loss provisions on the return on capital employed (ROCE) of deposit money banks in Nigeria.

# Function to create a box plot and histogram for the variables
def histogram_boxplot(data, feature, figsize=(15, 10), kde=False, bins=None):
    """
    Boxplot and histogram combined

    data: dataframe
    feature: dataframe column
    figsize: size of figure (default (15,10))
    kde: whether to show the density curve (default False)
    bins: number of bins for histogram (default None)
    """
    f2, (ax_box2, ax_hist2) = plt.subplots(
        nrows=2,  # Number of rows of the subplot grid= 2
        sharex=True,  # x-axis will be shared among all subplots
        gridspec_kw={"height_ratios": (0.25, 0.75)},
        figsize=figsize,
    )  # creating the 2 subplots
    sns.boxplot(
        data=data, x=feature, ax=ax_box2, showmeans=True, color="violet"
    )  # boxplot will be created and a triangle will indicate the mean value of the column
    sns.histplot(
        data=data, x=feature, kde=kde, ax=ax_hist2, bins=bins
    ) if bins else sns.histplot(
        data=data, x=feature, kde=kde, ax=ax_hist2
    )  # For histogram
    ax_hist2.axvline(
        data[feature].mean(), color="green", linestyle="--"
    )  # Add mean to the histogram
    ax_hist2.axvline(
        data[feature].median(), color="black", linestyle="-"
    )  # Add median to the histogram

# function to create labeled barplots
def labeled_barplot(data, feature, perc=False, n=None):
    """
    Barplot with percentage at the top

    data: dataframe
    feature: dataframe column
    perc: whether to display percentages instead of count (default is False)
    n: displays the top n category levels (default is None, i.e., display all levels)
    """

    total = len(data[feature])  # length of the column
    count = data[feature].nunique()
    if n is None:
        plt.figure(figsize=(count + 2, 6))
    else:
        plt.figure(figsize=(n + 2, 6))

    plt.xticks(rotation=90, fontsize=15)
    ax = sns.countplot(
        data=data,
        x=feature,
        palette="Paired",
        order=data[feature].value_counts().index[:n],
    )

    for p in ax.patches:
        if perc == True:
            label = "{:.1f}%".format(
                100 * p.get_height() / total
            )  # percentage of each class of the category
        else:
            label = p.get_height()  # count of each level of the category

        x = p.get_x() + p.get_width() / 2  # width of the plot
        y = p.get_height()  # height of the plot

        ax.annotate(
            label,
            (x, y),
            ha="center",
            va="center",
            size=12,
            xytext=(0, 5),
            textcoords="offset points",
        )  # annotate the percentage

    plt.show()  # show the plot
